# ➤ wl-divider

Thin line that groups content in lists and layouts.


[![-----------------------------------------------------](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/colored.png)](#properties)

## ➤ Properties

| Property   | Attribute  | Type       | Default     | Description                 |
|------------|------------|------------|-------------|-----------------------------|
| `role`     | `role`     | `AriaRole` | "separator" | Role of the backdrop.       |
| `vertical` | `vertical` | `boolean`  |             | Makes the divider vertical. |


[![-----------------------------------------------------](https://raw.githubusercontent.com/andreasbm/readme/master/assets/lines/colored.png)](#css-custom-properties)

## ➤ CSS Custom Properties

| Property          | Description                    |
|-------------------|--------------------------------|
| `--divider-color` | Color of the divider           |
| `--divider-size`  | Width or height of the divider |
